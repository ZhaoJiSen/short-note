JavaScript 是前端开发的基础语言，也是理解浏览器运行机制、异步编程、模块化和工程化的起点。

这部分笔记会从常用语法出发，逐步整理对象、数组、ES6+、原型链、`this`、Promise、`async/await` 等核心内容，尽量把零散知识串成一套能直接用于开发的理解框架。

如果你刚开始看这一部分，可以先从基础语法、对象、数组开始，再进入 ES6 与异步处理相关内容。
