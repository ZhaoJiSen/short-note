> [!IMPORTANT]
> 闭包是能捕获其定义时外部作用域值的匿名函数，被捕获的值会随闭包一起存在，按需借用或移动

## 基本使用

Rust 的闭包需要显示的通过 `||` 将内容包裹。通常情况下，闭包的参数和返回值类型会被编译器自动推断，如果是单独声明则需手动标注

:::rust-repl

```rust
fn main() {
    let sum = |a: i32, b: i32| a + b;

    println!("sum is {}", sum(1, 2));
}
```

:::rust-repl title="闭包的基本使用"

```rust
#[derive(Debug, PartialEq, Clone, Copy)]
enum ShirtColor {
    Blue,
    Red,
}

struct Inventory {
    shitrs: Vec<ShirtColor>,
}

impl Inventory {
    fn giveaway(&self, user_perference: Option<ShirtColor>) -> ShirtColor {
        user_perference.unwrap_or_else(|| self.most_stocked())
    }

    fn most_stocked(&self) -> ShirtColor {
        let mut num_red = 0;
        let mut num_blue = 0;

        for color in &self.shitrs {
            match color {
                ShirtColor::Red => num_red += 1,
                ShirtColor::Blue => num_blue += 1,
            }
        }

        if num_red > num_blue {
            ShirtColor::Red
        } else {
            ShirtColor::Blue
        }
    }
}

fn main() {
    let store = Inventory {
        shitrs: vec![ShirtColor::Blue, ShirtColor::Red, ShirtColor::Blue],
    };

    let user_profile1 = Some(ShirtColor::Red);
    let giveaway1 = store.giveaway(user_profile1);
    println!(
        "The user with preference {:?} gets {:?}",
        user_profile1, giveaway1
    );

    let user_profile2 = None;
    let giveaway2 = store.giveaway(user_profile2);
    println!(
        "The user with preference {:?} gets {:?}",
        user_profile2, giveaway2
    );
}
```
