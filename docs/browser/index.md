本章内容以「浏览器工作原理与实践」的核心脉络为主线，并结合工程视角补充了可直接落地的知识点。

## 学习地图

:::table
| 模块 | 关注点 | 你会得到什么 |
| --- | --- | --- |
| 宏观视角下的浏览器 | 进程模型、导航、渲染、跨标签通信 | 建立浏览器整体系统观 |
| V8 引擎工作原理 | 栈堆模型、GC、执行机制 | 能解释性能与内存问题 |
| 页面循环 | 事件循环、WebAPI、宏任务/微任务 | 能定位异步时序和卡顿来源 |
| 网络安全 | 同源策略、XSS、CSRF、CORS | 能设计前端安全基线 |
:::

:::details 说明
1. 本章中的流程图和机制图，尽量使用了本地化图片，避免外链失效。  
2. 新增内容包括：`CSRF`、`WebAPI`、`宏任务/微任务`、`TCP 三次握手`、`TLS 握手`、`TCP 四次挥手`、`CSR 与渲染路径`。
