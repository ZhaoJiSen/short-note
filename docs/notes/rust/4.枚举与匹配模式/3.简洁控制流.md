### if let 匹配

有时会遇到只有==一个模式==的值需要被处理，其它值直接忽略的场景，那么此时就可以使用 `if let` 来处理

```rust
if let Some(3) = v {
    println!("three");
}
```

::: details 用 `match` 匹配的写法

```rust
let v = Some(3);

match v {
  Some(3) => println!("three"),
  _ => (),
}
```

:::
