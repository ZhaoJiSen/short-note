数组本质上也是对象，但在响应式系统里它比普通对象更特殊。因为数组除了普通属性访问，还会涉及：

- 下标访问
- `length` 变化
- 新增、删除对迭代结果的影响
- `push / pop / shift / unshift / splice` 这类方法调用
- `includes / indexOf / lastIndexOf` 这类查询方法

所以对象上的那一套 `get / set / has / ownKeys / deleteProperty` 处理方式还不够，数组通常需要额外做一层专门处理。
