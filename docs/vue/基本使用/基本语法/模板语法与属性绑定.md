在 vue3.4 以上版本中，如果动态绑定的属性与数据同名，则可省略属性值的赋值

布尔值要额外注意，例如 disable
如果绑定的数据是真值或者空字符串，那么该布尔值为真
如果所绑定的数据是假值，则该布尔值会被忽略

如果想动态绑定多个值，这个时候可以直接绑定成一个对象

```vue
<template>
  <div v-bind="attrs"></div>
</template>

<script setup lang="ts">
const attrs = {
  id: 'wrapper',
  class: "wrapper-element",
}
</script>
```

## 模板沙盒化

模板中可以使用表达式，但是这些表达式都是沙盒化的，沙盒的主要意义是保证安全，虽然在模板中可以访问到全局对象，但由于沙盒的限制，只能访问到部分的全局对象
