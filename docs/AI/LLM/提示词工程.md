提示词工程（Prompt Engineering）是一门专注于 LLM 输入的学科，目的是用尽可能程序化、可复现的方式，把模型的输出稳定地推向理想结果。

要理解它为什么有效，得先抓住一件事：LLM 本质是个条件概率机器，它每一步做的都是 "在已经看到的全部 token 之后，下一个 token 最可能是什么"。提示词就是这段 "已经看到的全部 token"，是模型每一步预测所依赖的条件。所以写提示词，本质是在为模型的概率分布 **布置上下文**，把它从漫无边际的分布收敛到你想要的那个区域。

> [!NOTE]
> 这也划定了提示词工程的边界：它只能重新分配模型 **已经具备** 的能力和知识，无法凭空创造模型没学过的东西。当任务依赖模型不掌握的知识时，该上的是 RAG（检索增强）或微调，而不是继续堆提示词
